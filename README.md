# Learn Microsoft Excel for Data Analysis

Off-platform Codecademy course on data analysis fundamentals using Microsoft Excel, covering exploration, visualization, data handling, and real-world projects.

## Course overview

This course teaches core data-analysis skills in Excel through structured modules and hands-on projects. Topics include pivot tables, charting, data cleaning, and applied case studies.

## Course structure

### 1. Welcome to Learn Microsoft Excel for Data Analysis
Introductory module (currently placeholder).

### 2. Exploring Data
Learn to explore and summarize data in Excel.

**Topics:**
- Basic data exploration
- Summary statistics
- Pivot tables for aggregation and grouping

**Projects:**
- `1. Exploring Data/` — Foundational exploration exercises
- `2. Pivot Tables/` — Hands-on pivot table practice
- `3. Explore GDP in Excel/` — Real-world GDP dataset analysis

### 3. Visualizing Data
Create effective charts and visualizations in Excel.

**Topics:**
- Chart types and when to use them
- Formatting and styling visualizations
- Dashboard and multi-chart layouts

**Projects:**
- `1. Visualizing Data/` — Chart creation exercises
- `2. Visualizing Hotel Data in Excel/` — Hotel dataset visualization case study

### 4. Handling Data
Clean, transform, and prepare data for analysis.

**Topics:**
- Data validation
- Removing duplicates
- Finding and replacing
- Text functions (LEFT, RIGHT, MID, CONCATENATE, etc.)
- Conditional logic and error handling

**Projects:**
- `1. Handling Data/` — Data-cleaning exercises
- `2. Handling Hotel Data in Excel/` — Hotel dataset cleaning case study
- `3. Capstone Project/` — Comprehensive project combining exploration, visualization, and handling

### 5. Analyze Bitcoin Data in Excel
Final project analyzing cryptocurrency price data.

**Files:**
- `Bitcoin Price Analysis.xlsm` — Excel workbook with formulas and visualizations
- `OffPlatform-Data-Spreadsheets.xlsx` — Supporting data files
- `Analyze Bitcoin Data in Excel Codecademy.URL` — Course link

## Key skills covered

- **Data Exploration:** Sorting, filtering, basic statistics, pivot tables
- **Visualization:** Column charts, line charts, scatter plots, combo charts
- **Data Handling:** Cleaning, text functions, conditional formatting
- **Analysis:** RFM analysis, trend analysis, grouping and summarization
- **Excel Functions:** SUM, AVERAGE, IF, VLOOKUP, INDEX/MATCH, CONCATENATE, and more

## Files and datasets

- `Bitcoin Price Analysis.xlsm` — Bitcoin price analysis with historical data and visualizations
- `OffPlatform-Data-Spreadsheets.xlsx` — Sample datasets used throughout the course
- Multiple `.URL` files linking to Codecademy course pages

## Prerequisites

- Microsoft Excel (Windows or Mac) or compatible spreadsheet software (LibreOffice Calc, Google Sheets)
- Basic computer literacy

## How to use

1. Navigate to each module in numeric order (1 → 2 → 3 → 4 → 5).
2. Complete the lessons and exercises in each section.
3. Open the provided `.xlsm` and `.xlsx` files to follow along with examples.
4. Work through the projects within each module:
   - Follow guided instructions
   - Build formulas and charts
   - Clean and analyze data
5. Complete the Capstone Project in Module 4 to integrate all skills.
6. Use the Bitcoin analysis project as a reference for real-world application.

## Learning outcomes

After completing this course, you will be able to:
- Explore datasets using pivot tables and filters
- Create professional visualizations for reporting
- Clean and transform raw data
- Perform basic data analysis and summarization
- Build interactive dashboards
- Apply Excel functions to solve real business problems

## Tips for success

- Practice keyboard shortcuts (Ctrl+E for flash fill, Ctrl+T for tables, etc.)
- Use named ranges to make formulas more readable
- Create pivot tables from clean, well-organized data
- Always validate results—spot-check your calculations
- Organize your workbook with clear tabs and labels
